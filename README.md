# CowCare — One-Pager

Marketing-One-Pager für **CowCare**, das kamerabasierte KI-System für Milchviehbetriebe.
Statische Seite (ein `index.html`, keine Build-Tools), gehostet über GitHub Pages.

## Live

https://weconn3ct.github.io/cowcare/

## Anpassen

| Was | Wo |
|---|---|
| **Calendly-Link** (aktuell Platzhalter `calendly.com/weconn3ct`) | in `index.html` nach `calendly.com` suchen (3 Stellen: Nav, Hero, Kontakt) |
| Formular-Empfänger | `<form action="https://formsubmit.co/mentor.sadiku@weconn3ct.de">` — beim ersten Absenden schickt FormSubmit eine Bestätigungs-Mail an diese Adresse (einmal bestätigen, danach kommen Anfragen an) |
| Farben | CSS-Variablen am Anfang von `index.html` (`--forest`, `--lime`, `--accent`, …) |

## Design

Farben & Styling folgen dem CowCare-Pitchdeck (Forest-Green-Palette, Orange-Akzent, organische Blob-Formen).
Bilder wurden aus dem Pitchdeck extrahiert (`assets/`).
